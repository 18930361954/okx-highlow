- 品种/周期: BTC-USDT-SWAP / 6H
- 参数: `float_pct=0.003`, `tp_pct=0.008`, `sl_pct=0.015`
- 表现: 总收益 **+5126.00%**, 月化 +209.79%, MDD 37.7%, 胜率 77.5% (2822 笔), PF 1.50

配置片段(可粘进 config.yaml 的 pair_overrides 或 strategy 覆盖):
```yaml

pairs: [BTC-USDT-SWAP]
strategy:
  signal_bar: 6H   # 需要新增运行时支持,目前策略仍是 1D
  pair_overrides:
    BTC-USDT-SWAP:
      float_pct: 0.003
      tp_pct: 0.008
      sl_pct: 0.015
```
